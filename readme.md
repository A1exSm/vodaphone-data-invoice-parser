### About this project
This project is a simple tool which parses vodaphone phone bill PDF files and generates a report.
<br><br>
`Note: This is a functional project where I focused on parsing, any GUI and report generation is generative.`
### Setup
1. Simply clone the repository.
2. Pip install the dependencies.
3. Run gui.py
### Usage
* Select input files via the GUI (Files may also be dropped into the GUI).
* Select the output file.
* Click the `Generate Report` button.
* (Files may also be dropped into the GUI)
To run the project, you need to run the `main.py` file. It will open a GUI where you can select the PDF file and generate the report.
***
#### Dependencies
* pdfplumber
* reportlab
* matplotlib
* PyQt6
***
```bash
pip install pdfplumber reportlab matplotlib PyQt6
```
